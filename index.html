Mob programming

Mob programming is an approach to software development where the entire team works together on same task.
They can be in the same place using the same computer.
It can also be a remote mob programming session where the team collaborate through live share.
Even though the computer itself is not the same when working remotely,
the liveshare makes it possible for all participants to code in the same file.
One person in the team usually acts as a host and creates the files and starts the live share session.
Mob programming has many benefits. It helps the team to code with a higher pace and come up with more creative solutions.
